# StalNet

⚠️ **Aviso:** Este script é recomendado para uso no **Termux**.

Olá! Antes de usar o script, preciso deixar algo claro:  
este projeto foi criado com o objetivo de **promover educação e aprendizado em programação e informática**.

---

## 📦 Como usar o StalNet

### 1️⃣ Instalar dependências

Primeiro, execute o script responsável por instalar as bibliotecas necessárias:

```bash
python3 installGithubLibs.py
```

Aguarde até que todas as dependências sejam instaladas.

---

### 2️⃣ Executar o StalNet

Depois da instalação, você pode iniciar o script com:

```bash
python3 stalnet.py tiktok username
```

---

### 🧹 Dica útil

Você pode limpar a tela automaticamente usando o parâmetro `-c`:

```bash
python3 stalnet.py tiktok username -c clear
```

---

## 🚧 Status do projeto

O **StalNet ainda está em versão beta**, então atualmente apenas o **TikTok** é suportado.

Futuramente outras redes sociais poderão ser adicionadas.

---

## ⚠️ Uso responsável

Não utilize este script para invadir privacidade ou assediar pessoas.

O objetivo do projeto é **educacional** e voltado ao aprendizado de **programação e análise de dados públicos**.

---
